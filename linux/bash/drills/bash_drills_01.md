# Bash-Script Drills - Grundlagen & Syntax

---

## Aufgabe 1: Shebang ergänzen

Ergänze die fehlende Shebang-Zeile am Anfang des Scripts.

```bash
_____________
echo "Mein erstes Bash-Script!"
```

---

## Aufgabe 2: Echo-Befehl vervollständigen

Ergänze den fehlenden Befehl, um "Hallo Linux!" auszugeben.

```bash
_____ "Hallo Linux!"
```

---

## Aufgabe 3: Kommentar einfügen

Ergänze das fehlende Zeichen, um einen Kommentar zu erstellen.

```bash
__ Dies ist ein Kommentar
echo "Dieser Text wird ausgegeben"
```

---

## Aufgabe 4: Variable deklarieren

Ergänze die fehlenden Teile, um die Variable korrekt zu deklarieren.

```bash
name___"Max"
echo $name
```

---

## Aufgabe 5: Fehlerhafte Variablenzuweisung korrigieren

Korrigiere die fehlerhafte Variablenzuweisung.

```bash
alter = 25
echo $alter
```

## Aufgabe 6: Variable ausgeben

Ergänze das fehlende Zeichen, um den Wert der Variable auszugeben.

```bash
stadt="Berlin"
echo _____stadt
```

---

## Aufgabe 7: Zwei Befehle in einer Zeile

Ergänze das fehlende Zeichen, um beide Befehle in einer Zeile auszuführen.

```bash
echo "Erste Zeile"___ echo "Zweite Zeile"
```

## Aufgabe 8: Gültige Variablennamen erkennen

Welche der folgenden Variablendeklarationen sind gültig? Markiere die fehlerhaften und korrigiere sie.

```bash
meineVar=100        # _______
_geheim=200         # _______
var123=300          # _______
1variable=400       # _______
meine-var=500       # _______
test!wert=600       # _______
```

## Aufgabe 9: Variable mit Klammern

Ergänze die fehlenden Klammern, um "TestWert" auszugeben.

```bash
wort="Test"

```

## Aufgabe 10: Unterschied zwischen " und '

Was wird jeweils ausgegeben? Schreibe die erwartete Ausgabe daneben.

```bash
name="Anna"
echo "Hallo, $name!"     # Ausgabe: _______________
echo 'Hallo, $name!'     # Ausgabe: _______________
```
